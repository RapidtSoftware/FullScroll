FullScroll

is a vanilla JavaScript, full page slideshow library with smooth scrolling and arrow button control.
Downloador see me on Github.

Browser Support

Here is a browser compatibility table with the minimum browser versions that supports FullScroll.

Usage

Simply include the correct files, write the pages' HTML and invoke the fullscroll JavaScript function. 
<div class='fullscroll'>
	<h1>Hello</h1>
	<span>This is page 1</span>
</div>
<div class='fullscroll'>
	<h1>Page 2</h1>
	<span>This is page 2. Lorem impsum samet.</span>
</div>

<script src='path/to/fullscroll.min.js'></script>
<link rel='stylesheet' src='path/to/fullscroll.min.css'/>

<script>
	new FullScroll();
</script>

Features and Options

A list of currently available options and features. Note that more options will be added soon.
  • To use a specific document title for each page, just add a data-fullscroll-title attribute to your fullscroll page element and insert the title you want as the attribute's value.
  • You can change the selector of FullScroll pages by adding an option called selector to the FullScroll invocation function. For example: 
    new FullScroll({ selector: '#container > .my-page-element-selector' });
  • To do something each time that the page is changed, just add an options called selector. For example: 
    new FullScroll({ onChange: function(){ /* Do stuff here */ } });
  • To prevent spam pressing of arrow buttons, FullScroll adds each page to show to a queue and shows the next element in the queue when the current page's animation is finished.

Methods

A list of currently available methods. Note that more methods will be added soon.
  • .goto( index ) - Go to the page with the specfied index. For example: 
    new FullScroll().goto( 0 );
  • .next() - Go to the next page. For example: 
    new FullScroll().next();
  • .previous() or .prev() - Go to the previous page. For example: 
    new FullScroll().previous();
  • .current - Get the current page index. For example: 
    console.log( new FullScroll().current );

Events

A list of currently available events. Note that more events will be added soon.
  • onChange( element, direction | index ) - Fired each time that the page is changed. For example: 
    new FullScroll({ onChange: function(){ console.log( this.current ) } });
